# dotfile
環境毎の設定情報(mac, linux)

### install mac
```bash
# clone and install
git clone https://github.com/ztp1977/dotfile.git
cd dotfile/mac
install.sh

# vi起動
vi
```

### install linux
git clone https://github.com/ztp1977/dotfile.git
cd dotfile/linux
install.sh

### troubleshot
1. neovim plugin upgrade

```bash
rm -rf .config/nvim-dein
vim
```
